K-Aray Search

Due date: Thursday, 15 October 2015, 6:00 PM
Maximum number of files: 1
Type of work: Individual work

Implement the k-ary search. The idea is to find the search element in the given sorted array.

1. First divide the search space (array) into k partitions using (k-1) separators.

2. Choose a partition depending on the comparison of the separator.


The Procedure is as follows.

The given array is as follows

1,4,7,10,11,15,18,19,24,29,35,46,48,55,59,60,67,73,75,77,83,88,92,93,97,99

Here k value is 5 and Search Key is 77. 

So, the partitions are p0, p1, p2, p3, p4

p0 : 1,4,7,10,11

p1 : 15,18,19,24,29

p2 : 35,46,48,55,59

p3 : 60,67,73,75,77

p4 : 83,88,92,93,97,99

So, here separators are : 15,35,60,83 which are (k-1) separators. Now compare the search key with the separators.

Separators are : 15,35,60,83

Replicated Search Key is : 77,77,77,77

Lower than comparison result : -1,-1,-1,0


Here the vector "Lower than comparison result" sum is : -3 which means the search key is in partiion 3.  Repeat the process until the search key is found in the given array.


Now the partition 3 is as follows

60,67,73,75,77

Here k value is 5 and Search Key is 77. 

So, the partitions are p0, p1, p2, p3, p4

p0 : 60

p1 : 67

p2 : 73

p3 : 75

p4 : 77

So, here separators are : 15,35,60,83 which are (k-1) separators. Now compare the search key with the separators.

Separators are : 67,73,75,77

Replicated Search Key is : 77,77,77,77

Lower than comparison result : -1,-1,-1,-1


Here the vector "Lower than comparison result" sum is : -4 which means the search key is in partiion 4. If the partition size is less than k, then perform linear search on that partition.



Input:

The first line of the input contains the number of elements.

The second line of the input contians the k value.

The third line of the input contains the search element.

The fourth line of the input contains the array elements separated by commas (,)


Output:

The initial lines of the output contains the separators separated by a comma in each iteration followed by absolute of vector sum and the partition that you are choosing in each iteration, print them in separate lines.

The last line of the output should be, whether the search is successful or not. If the search is successful Print "Success" otherwise Print "Failure" without double quotes.


Note: If the elements in the partition is less than k (partition size), perform linear search on that partition.


Sample Input #1:

26

5

77

1,4,7,10,11,15,18,19,24,29,35,46,48,55,59,60,67,73,75,77,83,88,92,93,97,99


Sample Output #1:

15,35,60,83

p3

3

67,73,75,77

p4

4

Success
